# যিকির প্রো — GitHub Pages Deploy Guide

## ধাপ ১: GitHub Account খুলুন
👉 https://github.com/signup — সম্পূর্ণ ফ্রি

## ধাপ ২: Repository তৈরি করুন
1. github.com এ Login করুন
2. উপরে "+" বাটন → "New repository"
3. Repository name: **zikir** (ছোট হাতে)
4. Public রাখুন ✓
5. "Create repository" ক্লিক করুন

## ধাপ ৩: ফাইলগুলো Upload করুন
Repository-তে গিয়ে "uploading an existing file" ক্লিক করুন।
এই ৫টি ফাইল upload করুন:
- index.html
- manifest.json
- sw.js
- icon-192.png
- icon-512.png

তারপর "Commit changes" করুন।

## ধাপ ৪: GitHub Pages চালু করুন
1. Repository → "Settings" ট্যাব
2. বাম দিকে "Pages" ক্লিক করুন
3. Source: "Deploy from a branch"
4. Branch: "main" → "/" (root)
5. "Save" ক্লিক করুন

## ✅ আপনার App URL হবে:
https://আপনার-নাম.github.io/zikir

(৫ মিনিট পর activate হবে)

---

## Chrome এ Install করুন:
1. Phone এ Chrome খুলুন
2. আপনার URL লিখুন
3. কয়েক সেকেন্ড পর নিচে "Install" banner আসবে
4. Install বাটনে ট্যাপ করুন
5. Home Screen এ 🌙 আইকন দেখাবে!
